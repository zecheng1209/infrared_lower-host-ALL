# 红外通信系统 - 上位机与下位机模块

## 项目简介

本项目包含两个核心模块，实现基于红外通信的机械臂控制系统：

1. **Arm_infrared-host_Lower** - 机械臂控制上位机模块（STM32F407）
2. **U8T6--2** - 红外通信下位机模块（STM32F103）

两个模块通过红外链路进行无线通信，实现机械臂的远程控制和状态监控。

## 模块说明

### 1. Arm_infrared-host_Lower（上位机）

- **主控芯片**：STM32F407VETx
- **主要功能**：
  - 机械臂电机控制（支持RobStride系列）
  - CAN总线通信（双CAN配置）
  - FreeRTOS多任务管理
  - 红外上位机通信协议处理
  - PID位置/速度环控制

### 2. U8T6--2（下位机）

- **主控芯片**：STM32F103C8T6
- **主要功能**：
  - 38kHz红外载波通信
  - CAN总线与红外数据转发
  - CRC8校验和ACK确认机制
  - 超时重传和自发自收过滤

## 通信协议

### 帧类型定义

| 帧类型 | CAN ID格式 | 说明 |
|--------|-----------|------|
| CMD帧 | 0x1[module_id] | 命令帧 |
| DATA帧 | 0x2[module_id] | 数据帧 |
| ACK帧 | 0x3[module_id] | 确认帧 |

### 红外帧格式

```
[8字节数据][1字节CRC] = 9字节(72位)
```

## 硬件需求

### 上位机（Arm_infrared-host_Lower）
- STM32F407VETx开发板
- RobStride系列电机（03型/01型）
- RoboMaster电机（可选）
- CAN总线接口
- 红外通信模块

### 下位机（U8T6--2）
- STM32F103C8T6开发板
- 红外发射/接收模块（38kHz）
- CAN总线接口

## 软件需求

- Keil MDK-ARM开发环境
- STM32CubeMX
- FreeRTOS（上位机）
- STM32 HAL库

## 使用说明

### 1. 编译各模块
- 使用Keil MDK-ARM分别打开两个模块的项目
- 编译生成hex文件

### 2. 烧录程序
- 将上位机程序烧录到STM32F407
- 将下位机程序烧录到STM32F103

### 3. 硬件连接
- 上位机：连接电机和红外发射模块
- 下位机：连接红外接收模块和CAN总线

### 4. 系统启动
- 上位机启动后初始化电机和任务
- 下位机启动后等待红外数据
- 通过红外链路实现数据传输
