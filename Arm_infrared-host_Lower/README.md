# 机械臂控制系统（初版，之后又加入红外上位机的版本）

## 项目简介

这是一个基于STM32F407微控制器的机械臂控制系统，使用FreeRTOS实时操作系统进行任务管理，通过CAN总线与电机通信，实现对机械臂的精确控制。系统支持红外上位机通信，可实现远程控制和监控功能。（红外初始架构详见infrared的仓库）

## 硬件需求

- STM32F407VETx开发板
- RobStride系列电机（03型和01型）
- RoboMaster3508及2006系列电机
- CAN总线接口
- 红外通信模块
- 电源系统

## 软件需求

- Keil MDK-ARM开发环境
- STM32CubeMX
- FreeRTOS
- STM32F4xx HAL库

## 核心功能

### 1. 电机控制
- 支持RobStride系列电机的初始化和控制
- 实现位置环和速度环的PID控制
- 支持电机的使能和模式设置

### 2. CAN总线通信
- 双CAN总线配置（CAN1和CAN2）
- 电机数据的发送和接收
- 通信过滤和中断处理

### 3. 任务管理
- 基于FreeRTOS的多任务系统
- 电机驱动任务
- 红外上位机通信任务
- 任务优先级管理

### 4. 红外通信
- 红外上位机通信协议
- 远程控制和监控功能

### 5. 机械臂控制算法
- 关节角度计算
- 平滑运动控制
- 位置校准和复位功能

## 使用说明

### 1. 硬件连接
- 将电机通过CAN总线连接到STM32开发板
- 连接红外通信模块
- 确保电源供应稳定

### 2. 软件配置
- 使用STM32CubeMX生成基础代码
- 配置CAN总线参数
- 配置FreeRTOS任务

### 3. 编译和烧录
- 使用Keil MDK-ARM编译项目
- 通过JTAG或串口烧录到STM32开发板

### 4. 运行控制
- 系统启动后会自动初始化电机
- 通过红外上位机发送控制命令
- 监控电机状态和机械臂位置

## 代码架构

### 核心模块

#### 1. 任务初始化模块 (`Task_Init.c`)
- 负责系统初始化和任务创建
- 配置CAN总线和电机参数
- 启动各个控制任务

#### 2. 电机驱动模块 (`Run.c`)
- 实现电机的PID控制算法
- 处理电机状态数据
- 实现机械臂的运动控制

#### 3. 红外通信模块 (`infrared_host.c`)
- 实现与上位机的红外通信
- 处理控制命令和数据传输

#### 4. 电机控制库 (`RobStride2.h`)
- 提供RobStride电机的控制接口
- 实现电机的模式设置和使能功能

### 任务结构

| 任务名称 | 优先级 | 栈大小 | 功能描述 |
|---------|--------|--------|----------|
| Motor_Drive | 4 | 628 | 电机驱动和控制 |
| IR_Host_Task | 3 | 512 | 红外上位机通信 |
| Motor_reset | 4 | 300 | 电机复位（可选） |
| MotorSendTask | 4 | 128 | 数据发送到PC（可选） |
| MotorRecTask | 4 | 200 | 从PC接收数据（可选） |

## 核心API

### 电机控制API

#### `RobStrideInit`
初始化RobStride电机
- **参数**：电机结构体、CAN句柄、电机ID、电机类型
- **返回值**：无

#### `RobStrideSetMode`
设置电机工作模式
- **参数**：电机结构体、工作模式
- **返回值**：无

#### `RobStrideEnable`
使能电机
- **参数**：电机结构体
- **返回值**：无

### PID控制API

#### `PID_Init_Pos`
初始化位置PID控制器
- **参数**：关节结构体、KP、KI、KD、积分限幅、输出限幅
- **返回值**：无

#### `PID_Init_Vel`
初始化速度PID控制器
- **参数**：关节结构体、KP、KI、KD、积分限幅、输出限幅
- **返回值**：无

### 任务管理API

#### `Task_Init`
初始化系统任务
- **参数**：无
- **返回值**：无

#### `Motor_Drive`
电机驱动任务
- **参数**：任务参数
- **返回值**：无

## 注意事项

1. **电机参数调整**：根据实际机械臂结构调整PID参数和位置偏移值
2. **CAN总线配置**：确保CAN总线的波特率和过滤配置正确
3. **电源供应**：确保电机和控制器有足够的电源供应
4. **安全保护**：在机械臂运动范围内设置限位保护
5. **调试模式**：建议在调试时降低电机速度，确保安全
